CRÉATION DE SITE WEB AVEC HTML5 & CSS3

Contenu de la formation

Le HTML5 et le CSS3 sont les dernières versions des principaux langages Web validés par le World Wide Web Consortium (W3C). Complémentaires, le HTML permet d’afficher le contenu des pages web et le CSS vise à décrire la présentation de ces contenus. Les deux sont associés dans le cadre de leur développement pour le web.

Aujourd’hui, pour créer un site web professionnel, il est désormais indispensable de maîtriser les bases de la programmation HTML5 CSS3.

Vous vous intéressez au développement de site web mais vous ne savez pas comment vous y prendre ? Vous pensez devoir faire appel à un prestataire extérieur ? Pas nécessairement !

La formation VISIPLUS academy 2 jours vous apprendra à créer des pages Web en utilisant HTML5 et CSS3, les deux langages à la base de tous les sites web.
